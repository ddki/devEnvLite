| Permission | Description |
|------|-----|
|`allow-get-theme`|Enables the get_theme command without any pre-configured scope.|
|`deny-get-theme`|Denies the get_theme command without any pre-configured scope.|
|`allow-set-theme`|Enables the set_theme command without any pre-configured scope.|
|`deny-set-theme`|Denies the set_theme command without any pre-configured scope.|
